# 🎮 Tic-Tac-Toe AI (Minimax + Easy Mode)

A terminal-based Tic-Tac-Toe game built in Python for the *CodSoft AI Internship – Task 2*.

This version includes:
- 🧠 AI using Minimax algorithm (Hard mode)
- 🎲 Easy mode with random moves
- 🎯 Score tracking (Best of 3 match)
- ✔ Clean board with 1–3 input indexing

## 🔧 Features
- You play as 'X', AI plays as 'O'
- Choose difficulty: Easy or Hard
- Unbeatable AI in Hard Mode (Minimax)
- Clear scoreboard and round summary

## 👨‍💻 How to Run

```bash
python tic_tac_toe_ai.py
